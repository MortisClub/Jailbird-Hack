## 📦 Installation

### Method 1: Direct Execution (Recommended)

<div align="center">

![Installation Step 1](screenshots/install-step1.png)
*Шаг 1: Откройте Roblox и присоединитесь к Jailbird*

![Installation Step 2](screenshots/install-step2.png)
*Шаг 2: Откройте ваш executor*

</div>

1. **Open Roblox** and join **Jailbird** game
2. **Open your executor** (Synapse X, KRNL, Fluxus, etc.)
3. **Paste the loadstring:**
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/MortisClub/Jailbird-Hack/main/obf_4gPh7M0718NQMMjbiEeuIIoW00gjUfRcuy41IpKK3LqFWU6eDbheeHfmj1T6Q7uf.lua"))()
```
4. **Execute** and wait for the GUI to load (3-5 seconds)
5. Press **RightShift** to open menu

<div align="center">

![Installation Complete](screenshots/install-complete.png)
*Успешная загрузка - меню готово к использованию!*

</div>

### Method 2: AutoExecute

1. Download the script file from [Releases](https://github.com/MortisClub/Jailbird-Hack/releases)
2. Place it in your executor's `autoexec` folder
3. Restart executor
4. Script will auto-load on game join

### Method 3: Manual Load

1. Download the raw script
2. Copy the entire content
3. Paste into your executor
4. Click Execute

### Script Won't Load

```
Symptom: No GUI appears after execution

Checklist:
□ Check internet connection
□ Verify executor supports HttpGet
□ Try alternative loadstring
□ Restart executor completely
□ Rejoin game
□ Check if executor is up to date
□ Disable other running scripts
□ Try on fresh server

Solution Steps:
1. Close Roblox
2. Restart executor
3. Join smaller server (<10 players)
4. Execute script
5. Wait 15 seconds
6. Press RightShift
```

### Low FPS / Performance Issues

```
Symptom: Game lags or stutters with script running

Optimization Steps:
1. Disable Chams (biggest performance hit)
2. Reduce ESP max distance to 1000
3. Disable custom crosshair
4. Lower FOV circle quality (fewer sides)
5. Disable watermark
6. Turn off unused features

In-Game Settings:
- Graphics Quality: Manual → 1-5
- Render Distance: Lower
- Shadows: Off
- Particles: Off

Expected FPS:
Low-end: 30-45 FPS
Mid-range: 45-60 FPS
High-end: 60+ FPS (capped)
```

### Detected / Kicked / Banned

```
Symptom: Account banned or kicked from game

Prevention:
□ Use ALT accounts only

□ Don't use obvious speed (>40)
□ Don't use rage aimbot in populated servers
□ Take 10-minute breaks every hour

□ Don't report other players while cheating

Legit mode: ~5-10%
Balanced: ~20-30%
Rage mode: ~60-80%
```

### Aimbot Jittering / Shaking

```
Symptom: Aimbot aim is unstable or shaky

Solutions:
1. Increase smoothness (8 → 12-15)
2. Reduce shake amount (2.0 → 0.5)
3. Disable aim drift
4. Lower prediction value (0.135 → 0.05)
5. Check FOV lock is enabled
6. Ensure stable FPS (60+)
7. Disable other scripts

Settings to Try:
Smoothness: 12-15
Shake: 0.3-0.8
Drift: Disabled
Prediction: 0.05 or OFF
FOV Lock: Enabled
```

### ESP Flickering / Disappearing

```
Symptom: ESP elements flash or vanish

Fixes:
1. Check max distance setting
2. Disable team check (testing)
3. Increase ESP update rate
4. Rejoin less crowded server
5. Disable Chams if using
6. Ensure stable internet connection

If Only Specific Players Missing:
- They might be in vehicles
- Check team check is configured right
- Verify they're not too far
- Ensure they're not dead (Health > 0)
```

### FOV Keeps Resetting

```
Symptom: Camera FOV returns to default

Solution:
✅ Enable "FOV Changer + Lock" in Visual tab
✅ Ensure FOV Lock feature is ON
✅ Set desired FOV value
✅ Don't manually adjust FOV in Roblox settings

This happens because:
- Weapon ADS animations reset FOV
- Some game scripts force default FOV
- FOV Lock prevents this by constantly updating

If still resetting:
- Rejoin and re-execute
- Check no other scripts conflict
- Verify "FOV Lock" shows as enabled
```
